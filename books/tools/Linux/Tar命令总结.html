Tar命令总结

tar 命令总结

选项

  • -c

    create 创建归档文件

  • -x

    extract 提取归档文件

  • -v

    verbose 显示详情

  • -f

    file 操作文件.

  • -t

    显示归档文件内容

  • -j

    调用 bzip2

  • -z

    调用 gzip

  • -r

    追加/更新文件(夹)到已经存在的归档文件

  • -wildcards

    通配符

  • -C

    指定目录

注意：

  • -f 选项是必须的，用来指定文件，没有的话命令将无法执行执行
